# Wander-Movies: Responsive Movie Dashboard

Welcome to Wander-Movies, a responsive movie dashboard designed to provide a sleek and intuitive interface for browsing movies and series. This project leverages modern web development techniques to ensure an engaging user experience across various devices.

## Table of Contents
- [Features](#features)
- [Technologies](#technologies)
- [Setup](#setup)
- [Usage](#usage)
- [Contributing](#contributing)
## Features
- **Responsive Design:** Optimized for both desktop and mobile viewing.
- **Dynamic Content:** Browse through different categories including Movies, Series, and Favorites.
- **Interactive Elements:** Includes a search bar, notification icons, and user profile management.
- **Swiper Integration:** Smooth, interactive carousels for showcasing movies and new releases.

## Technologies
- **HTML5 & CSS3:** For structuring and styling the application.
- **JavaScript:** For dynamic functionality and interactivity.
- **Swiper.js:** For implementing responsive carousels.
- **RemixIcon:** For iconography.

## Setup
To run this project locally, follow these steps:

1. **Clone the repository:**
    ```sh
    git clone https://github.com/your-username/wander-movies-dashboard.git
    ```

2. **Navigate to the project directory:**
    ```sh
    cd wander-movies-dashboard
    ```

3. **Open `index.html` in your preferred web browser.**

## Usage
- **Navigation:** Use the sidebar to navigate between Home, Movies, Series, and Favorites.
- **Search:** Use the search bar to find specific movies.
- **Carousels:** Browse through movie suggestions using the interactive carousels.
- **User Profile:** Manage your profile through the account section in the navigation menu.
